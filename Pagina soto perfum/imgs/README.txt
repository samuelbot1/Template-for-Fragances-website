INSTRUCCIONES PARA AGREGAR IMÁGENES DE PERFUMES
================================================

1. Descarga imágenes de botellas de perfume desde Google Images, Unsplash, etc.

2. Guarda las imágenes en esta carpeta (imgs/) con el siguiente formato de nombre:

   Ejemplo: "one-million-parfum.jpg"
   
   Regla: 
   - Nombre del perfume + marca en minúsculas
   - Reemplaza espacios con guiones (-)
   - Solo caracteres alfanuméricos y guiones
   - Extensión: .jpg, .png, .webp

   Ejemplos de nombres correctos:
   - one-million-parfum.jpg
   - versace-eros.jpg
   - lattafa-khamrah.jpg
   - ariana-grande-cloud.jpg
   - bulgari-man-in-black.jpg
   - dolce-gabana-light-blue.jpg

3. El sistema automáticamente buscará la imagen correspondiente cuando se abra la tarjeta del perfume.

4. Si no encuentra la imagen, mostrará un fallback elegante con el logo dorado.

¡Listo! Las imágenes aparecerán al hacer click en las tarjetas de perfumes.
